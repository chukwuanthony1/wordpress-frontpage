Simple Setup Instructions 1) download the repo from the git and run docker compose up 2) create a wordpress project and login to the backend and activate the theme (Anthony Chukwu) 3) Craete a page (e.g homepage) and setup reading as the homepage in settings 4) Create the menu in the menu section 5) Add the Site title and the logs in the appearance and customize 6) To compile the static assets we used gulp (cd /wp-content/themes/anthony-chukwu) 7) run npm install to install gulp and to compile assets, run gulp watch
